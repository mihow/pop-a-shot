# pop-a-shot
Hardware &amp; software for 4 lane pop-a-shot machine

## Display pins

- Clock pin: 42
- Latch pin: 44
- Data pin: 43

## Button pins (digital inputs):

4 pin connectors

- B1: 36 37 38 GND
- B2: 30 31 32 GND
- B3: 33 34 35 GND
- B4: 39 40 41 GND
 
## Hoop Sensor pins (analog inputs):

3 pin connectors

- Hoop 1: PWR A2 GND
- Hoop 2: PWR A3 GND
- Hoop 3: PWR A5 GND
- Hoop 4: PWR A4 GND
